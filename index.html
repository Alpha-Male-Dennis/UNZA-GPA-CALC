How to use this calculator

  1. Count how many courses you have in each grade category
  2. Enter the counts in the appropriate fields below
  3. Half courses are worth 15 credits each
  4. Full courses are worth 30 credits each
  5. Click "Calculate GPA" to see your results

Note: This calculator uses the UNZA grading system. Leave fields blank if you have no courses in that category.
